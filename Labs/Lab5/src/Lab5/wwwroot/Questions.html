  1. List and describe 3 types of static content that can be used in MVC Core
    Images(.jpg, etc.), Scripts(.js), and StyleSheets(.css)
  2. Where does static content belong in an MVC Core application?
    In the "wwwroot" folder.
  3. Explain how to link to a static page from a controller and/or view.
    To link to a static page, the path specified in the "href" tag needs to start with a tilde, assuming that the static file is found in the root folder.
